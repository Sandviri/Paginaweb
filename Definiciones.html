Diferentes autores han planteado múltiples definiciones de los Sistemas de Información(SI), algunas de ellas son:


  • "SI es la combinación de recursos humanos y materiales que resultan de las operaciones de almacenar, recuperar y usar datos con el propósito de una gestión eficiente en las operaciones de las organizaciones"Samuelson
  • Se asume la visión integradora que considera que “la organización genera el SI y ambas se integran para formar un todo. Es decir una agrupación de componentes que trabajan juntos para alcanzar un objetivo común”Chaín
  • “Un SI consiste en un conjunto de personas, procesos, datos, modelos, tecnología y un lenguaje parcialmente formalizado, que forman una estructura cohesiva que sirve a algunos propósitos organizacionales o funciones” Davis y Olson

    • El objetivo de cualquier SI es conectar a un usuario con una fuente de información que necesita para satisfacer sus necesidades.




    Otras definiciones para resaltar




    "Un sistema para convertir datos procedentes del exterior o del interior del mismo en información y para comunicar esta en forma apropiada a los directivos de todos los niveles y de todas las funciones para facilitar la toma de decisiones efectivas y oportunas para la planeación, dirección y control de las actividades de las cuales ellos son responsables”Lucey


    "Conjunto de componentes interrelacionados que recolectan (o recuperan), procesan, almacenan y distribuyen información para apoyar la toma de decisiones y el control en una organización"Laudon
